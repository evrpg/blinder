# Roadmap

> Narrative companion to `blinder/feature_list.json` (which stays canonical).
> The **Planner** maintains this: one section per epic, one row per feature, with
> the *why*. It records how an initiative was split into features — it is not a
> spec. Details and decisions live in each feature's `blinder/specs/<id>-<name>/`.
